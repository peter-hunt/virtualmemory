# Virtual Memory

Virtual Memory is a Python project for computer memory simulation

## Installation

Use git to install Virtual Memory.

```bash
git install https://github.com/peter-hunt/virtualmemory.git
```

## Usage

```python
from virtualmemory import Bit

print(Bit(0))
print(Bit(1))
```
